Pizza API gives developers everything they need to embed a full pizza ordering workflow into any application. Whether you're building a consumer storefront, a corporate lunch portal, or an automated ordering bot, Pizza API handles the menu, orders, delivery tracking, and customer data so you can focus on the experience.

## Key Features

Pizza API is designed to cover every stage of the ordering lifecycle — from browsing the menu to doorstep delivery — with a clean, predictable REST interface.

<CardGroup cols={3}>
  <Card title="Menu Browsing" icon="list">

  </Card>
  <Card title="Custom Orders" icon="sliders">

  <Card title="Delivery Tracking" icon="truck">

## Base URL

All Pizza API endpoints are served from the following base URL:

```
https://api.pizzaapi.com/v1
```

<Note>
  Every request must be made over **HTTPS** and must include a valid API key in the `Authorization` header. Requests over plain HTTP or without a key will be rejected with a `401 Unauthorized` response.
</Note>
