description: "GET /menu — Devuelve todos los artículos disponibles. Filtra por categoría, disponibilidad o término de búsqueda. Devuelve id, nombre, tamaños e ingredientes."
---

Obtén el menú completo o filtra los resultados por categoría, disponibilidad o palabra clave. Este endpoint está paginado y devuelve una lista de objetos de artículos del menú junto con metadatos del resultado completo.

```text
GET https://api.pizzaapi.com/v1/menu
```

## Parámetros de consulta

<ParamField query="category" type="string">
  Filtra por categoría del menú. Valores aceptados: `pizza`, `side`, `drink`.
</ParamField>

<ParamField query="available" type="boolean">
  Ponlo en `true` para devolver solo artículos disponibles actualmente.
</ParamField>

<ParamField query="search" type="string">
  Busca artículos por nombre o descripción. Realiza una coincidencia parcial insensible a mayúsculas.
</ParamField>

<ParamField query="limit" default="20" type="integer">
  Número máximo de artículos por página. Mínimo: `1`, máximo: `100`.
</ParamField>

<ParamField query="offset" default="0" type="integer">
  Número de artículos a omitir antes de empezar a devolver resultados. Úsalo con `limit` para paginación.
</ParamField>

## Ejemplo de solicitud

<CodeGroup>

```bash cURL
curl https://api.pizzaapi.com/v1/menu?category=pizza&available=true \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  "https://api.pizzaapi.com/v1/menu?category=pizza&available=true",
  {
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
    },
  }
);

const menu = await response.json();
console.log(menu);
```

</CodeGroup>

## Campos de la respuesta

<ResponseField name="data" type="array">
  Array de objetos de artículos del menú que coinciden con la consulta.

  <Expandable title="Campos del objeto de artículo">
    <ResponseField name="data[].id" type="string">
      Identificador único del artículo.
    </ResponseField>

    <ResponseField name="data[].name" type="string">
      Nombre visible del artículo.
    </ResponseField>

    <ResponseField name="data[].description" type="string">
      Descripción corta con los ingredientes clave.
    </ResponseField>

    <ResponseField name="data[].category" type="string">
      Categoría a la que pertenece. Uno de `pizza`, `side` o `drink`.
    </ResponseField>

    <ResponseField name="data[].sizes" type="array">
      Combinaciones de tamaño y precio disponibles.

      <Expandable title="Campos del objeto de tamaño">
        <ResponseField name="data[].sizes[].size" type="string">
          Etiqueta de tamaño: `small`, `medium` o `large`.
        </ResponseField>

        <ResponseField name="data[].sizes[].price" type="number">
          Precio en USD para este tamaño.
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="data[].available_toppings" type="array">
      Array de IDs de ingredientes que se pueden añadir.
    </ResponseField>

    <ResponseField name="data[].image_url" type="string">
      URL de la imagen del artículo.
    </ResponseField>

    <ResponseField name="data[].is_available" type="boolean">
      Si el artículo está disponible para pedir actualmente.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer">
  Número total de artículos que coinciden antes de la paginación.
</ResponseField>

<ResponseField name="limit" type="integer">
  Valor de `limit` usado en esta respuesta.
</ResponseField>

<ResponseField name="offset" type="integer">
  Valor de `offset` usado en esta respuesta.
</ResponseField>
