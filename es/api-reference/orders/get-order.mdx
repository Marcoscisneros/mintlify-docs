---

Recupera los detalles completos de un pedido, incluidos su estado actual, todas las líneas con precios resueltos, la dirección de entrega y la hora estimada de entrega. Usa este endpoint para pantallas de confirmación o para consultar actualizaciones de estado.

```text
GET https://api.pizzaapi.com/v1/orders/{id}
```

## Parámetros de ruta

<ParamField path="id" type="string" required>
  Identificador único del pedido a recuperar (p. ej. `order_55ABCD112233`).
</ParamField>

## Ejemplo de solicitud

<CodeGroup>

```bash cURL
curl https://api.pizzaapi.com/v1/orders/order_55ABCD112233 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const orderId = "order_55ABCD112233";

const response = await fetch(
  `https://api.pizzaapi.com/v1/orders/${orderId}`,
  {
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
    },
  }
);

const order = await response.json();
console.log(order);
```

</CodeGroup>

## Campos de la respuesta

<ResponseField name="id" type="string">
  Identificador único del pedido.
</ResponseField>

<ResponseField name="status" type="string">
  Estado actual del pedido. Valores posibles:

  | Valor | Descripción |
  | --- | --- |
  | `pending` | Pedido realizado, sin confirmar por la cocina. |
  | `confirmed` | La cocina aceptó el pedido. |
  | `preparing` | El pedido se está preparando. |
  | `out_for_delivery` | El pedido está en camino. |
  | `delivered` | El pedido fue entregado. |
  | `cancelled` | El pedido fue cancelado. |
</ResponseField>

<ResponseField name="customer_id" type="string">
  ID del cliente que realizó el pedido.
</ResponseField>

<ResponseField name="subtotal" type="number">
  Suma de los totales de línea antes de impuestos, en USD.
</ResponseField>

<ResponseField name="tax" type="number">
  Importe de impuestos calculado, en USD.
</ResponseField>

<ResponseField name="total" type="number">
  Total final del pedido incluyendo impuestos, en USD.
</ResponseField>

<ResponseField name="created_at" type="string">
  Marca ISO 8601 de creación del pedido.
</ResponseField>

<ResponseField name="estimated_delivery_time" type="string">
  Marca ISO 8601 de la ventana estimada de entrega.
</ResponseField>
