This guide walks you through the entire Pizza API flow from scratch. By the end you'll have authenticated successfully, browsed the live menu, placed a real order, and retrieved its delivery status — all with plain HTTP calls. No SDK required.

<Steps>

  <Step title="Get your API key">
    [Sign up for a Pizza API account](https://dashboard.pizzaapi.com/signup) if you haven't already. Once you're in the dashboard, navigate to **Settings → API Keys** and copy your key. Keep it somewhere safe — you'll use it in every request.
  </Step>

  <Step title="Make your first request">
    Verify your key is working by calling the menu endpoint. Replace `YOUR_API_KEY` with the key you copied above.

    <CodeGroup>

    ```bash curl
    curl https://api.pizzaapi.com/v1/menu \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```

    ```javascript JavaScript
    const response = await fetch("https://api.pizzaapi.com/v1/menu", {
      headers: {
        Authorization: "Bearer YOUR_API_KEY",
      },
    });

    const menu = await response.json();
    console.log(menu);
    ```

    ```python Python
    import requests

    response = requests.get(
        "https://api.pizzaapi.com/v1/menu",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
    )

    print(response.json())
    ```

    </CodeGroup>

    A `200 OK` response confirms your key is valid and you're ready to move on.
  </Step>

  </Step>

  <Step title="Place an order">
    Send a `POST` request to `/orders` with the customer, items, and delivery address in the request body.

    <CodeGroup>

    ```bash curl
    curl -X POST https://api.pizzaapi.com/v1/orders \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "customer_id": "cus_abc123",
        "items": [
          {
            "menu_item_id": "pizza_margherita",
            "size": "large",
            "quantity": 1,
            "toppings": ["extra_cheese", "basil"]
          }
        ],
        "delivery_address": {
          "street": "123 Main St",
          "city": "San Francisco",
          "state": "CA",
          "zip": "94102"
        }
      }'
    ```

    ```javascript JavaScript
    const response = await fetch("https://api.pizzaapi.com/v1/orders", {
      method: "POST",
      headers: {
        Authorization: "Bearer YOUR_API_KEY",
        "Content-Type": "application/json",
      },
      body: JSON.stringify({
        customer_id: "cus_abc123",
        items: [
          {
            menu_item_id: "pizza_margherita",
            size: "large",
            quantity: 1,
            toppings: ["extra_cheese", "basil"],
          },
        ],
        delivery_address: {
          street: "123 Main St",
          city: "San Francisco",
          state: "CA",
          zip: "94102",
        },
      }),
    });

    const order = await response.json();
    console.log(order);
    ```

    ```python Python
    import requests

    payload = {
        "customer_id": "cus_abc123",
        "items": [
            {
                "menu_item_id": "pizza_margherita",
                "size": "large",
                "quantity": 1,
                "toppings": ["extra_cheese", "basil"],
            }
        ],
        "delivery_address": {
            "street": "123 Main St",
            "city": "San Francisco",
            "state": "CA",
            "zip": "94102",
        },
    }

    response = requests.post(
        "https://api.pizzaapi.com/v1/orders",
        json=payload,
        headers={"Authorization": "Bearer YOUR_API_KEY"},
    )

    print(response.json())
    ```

    </CodeGroup>

    A successful response returns the new order object:

    ```json Sample POST /orders response

      "status": "pending",
      "customer_id": "cus_abc123",
      "items": [
        {

          "size": "large",
          "quantity": 1,
          "toppings": ["extra_cheese", "basil"]
        }
      ],
      "delivery_address": {
        "street": "123 Main St",

      },
      "estimated_delivery_time": "2024-07-15T19:45:00Z",
      "created_at": "2024-07-15T19:15:00Z"
    }
    ```

    Save the `id` value (e.g. `ord_7x9kLmQ3`) — you'll use it to track the delivery.
  </Step>

  <Step title="Track your order">
    Poll `GET /delivery/{order_id}/status` at any time to get live delivery information.

    ```bash curl
    curl https://api.pizzaapi.com/v1/delivery/ord_7x9kLmQ3/status \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```

    ```json Sample delivery status response
    {
      "order_id": "ord_7x9kLmQ3",
      "status": "in_transit",
      "driver": {
        "name": "Alex R.",
        "phone": "+14155550192",
        "vehicle": "Honda Civic — 7KZP482"
      },
      "estimated_arrival": "2024-07-15T19:45:00Z",
      "last_updated": "2024-07-15T19:32:00Z"
    }
    ```

    Possible `status` values are `pending`, `preparing`, `in_transit`, and `delivered`.
  </Step>

</Steps>
