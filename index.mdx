## Get started in 4 steps

<Steps>
  <Step title="Get your API key">
    Sign up at the [dashboard](https://dashboard.pizzaapi.com/signup) and generate an API key from your account settings.
  </Step>
  <Step title="Authenticate your requests">
    Pass your key in the `Authorization` header: `Authorization: Bearer YOUR_API_KEY`
  </Step>
  <Step title="Browse the menu">
    Call `GET /menu` to fetch available pizzas, toppings, and sizes.
  </Step>
  <Step title="Place your first order">
    Use `POST /orders` with your chosen items to create a live order.
  </Step>
</Steps>

<Note>
  All API requests use HTTPS and must include a valid API key. See [Authentication](/authentication) for details.
</Note>
